# Xscreeps

#### 介绍
Screeps游戏个人代码，Screeps是一款硬核大型多人在线编程游戏，目前在steam有上架。

#### 功能
1.  孵化模块
2.  任务角色制系统
4.  lab的boost、合成
5.  主动防御、战争模块
6.  寻路、跨shard (我们是超时空军团)
7.  商业行为
